* Signing
In the local repository you would do:
#+begin_src sh
# Specify signing key
git config user.signingKey ~/.ssh/<your SSH key>.pub

# Tell git to sign using a SSH key
git config gpg.format ssh

# Always sign commits
git config commit.gpgSign true
#+end_src
